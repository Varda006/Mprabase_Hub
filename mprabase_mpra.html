MPRA Base Enhancer Elements (hg38)

This track displays enhancer elements assayed using Massively Parallel Reporter Assays (MPRA), STARR-seq, and related reporter technologies across multiple published studies. All coordinates are provided in the GRCh38/hg38 genome assembly.

Data Aggregation

This track aggregates elements from multiple independent MPRA experiments. Where necessary, hg19 coordinates were lifted over to hg38 using UCSC liftOver. Unmapped intervals were excluded.

Each element represents a tested regulatory sequence from a published experiment.

Definition of an "Experiment"

An experiment is defined as a unique combination of:

  • Cell line / tissue
  • Assay type (e.g., lentiMPRA, plasmidMPRA, STARR-seq)
  • PubMed ID (PMID)

All normalization and ranking procedures are performed independently within each experiment.

Score Normalization and Percentile Calculation

For each experiment:

  • Elements are sorted by raw activity score.
  • A percentile rank (0–100) is computed relative to other elements within the same experiment.
  • A UCSC-compatible scaled score (0–1000) is derived within each experiment to support browser display.

Both percentile rank and scaled score are calculated independently within each experiment. This avoids cross-experiment distortion due to differing assay dynamic ranges (e.g., MPRA vs STARR-seq).

Color Coding Scheme

Elements are colored according to percentile rank within experiment:

  • Blue: Percentile < 50
  • Orange: 50 ≤ Percentile < 75
  • Red: Percentile ≥ 75

Color bins are determined using percentile rank within each experiment. Coloring reflects relative regulatory activity strength within that experiment.
