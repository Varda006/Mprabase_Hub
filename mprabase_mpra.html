Description

This track displays regulatory elements assayed using Massively Parallel Reporter Assays (MPRA), STARR-seq, and related high-throughput reporter technologies across multiple published studies. Coordinates correspond to the human GRCh38 (hg38) genome assembly.

Massively Parallel Reporter Assays (MPRAs) enable quantitative testing of thousands of candidate regulatory DNA sequences in parallel by linking each sequence to a reporter gene and measuring transcriptional output using sequencing. These assays are widely used to identify and characterize enhancer activity across cell types and experimental conditions.

MPRA Base is a curated database of experimentally tested cis-regulatory elements derived from MPRA and related reporter assays. The database integrates datasets from multiple studies, assay platforms, and biological systems while preserving experiment-level resolution.

Many MPRA studies test synthetic regulatory sequences or designed oligonucleotide libraries that do not correspond to a unique genomic location. Because genome browser tracks require genomic coordinates, MPRA Base only includes elements derived from genomic fragments that can be mapped to the reference genome. As a result, some MPRA publications may not appear in this browser track if their experiments were performed exclusively on synthetic libraries without genomic coordinates.

Display Conventions

An experiment in MPRA Base is defined as a unique combination of:

  • Cell line or tissue
  • Assay type
  • Publication (PMID)

Each row in the track represents a genomic fragment tested within a specific experiment.

The same genomic region may appear multiple times if it was tested in different experiments (for example different cell types, assay platforms, or publications).

Track colors indicate percentile rank of the mean raw activity score within each experiment.

  • Blue – Percentile < 50
  • Orange – Percentile 50–74
  • Red – Percentile ≥ 75

Display color is determined by percentile-based RGB bins. Score-based shading is not used.

Methods

Within each experiment, genomic fragments may be measured multiple times through technical replicates or overlapping reporter constructs. To obtain a stable estimate of enhancer activity, replicate measurements corresponding to the same genomic fragment within each experiment were aggregated by computing the mean raw activity score.

To provide additional information about replicate variability, all individual replicate activity scores are recorded and displayed directly in the browser tooltip as a comma-separated list. This allows users to inspect the full distribution of replicate measurements for each element.

The original dataset contained 211,053 replicate-level measurements. After aggregation of replicate observations corresponding to the same genomic fragment within an experiment, the final database contains 41,275 unique experiment-level genomic elements.

Analysis of replicate measurements across experiments showed low variance in activity scores, indicating strong agreement between replicate measurements. Because of this low variability, the mean raw activity score provides a robust representation of enhancer activity for each tested element.

Elements are ranked by mean raw activity score within each experiment. A percentile rank (0–100) is computed independently for each experiment.

All ranking procedures are performed independently per experiment to avoid cross-study distortions caused by differing assay dynamic ranges.
