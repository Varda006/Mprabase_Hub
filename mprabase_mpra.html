MPRA Base Enhancer Elements (hg38)

This track displays regulatory elements assayed using Massively Parallel Reporter Assays (MPRA), STARR-seq, and related high-throughput reporter technologies across multiple published studies. Coordinates shown here correspond to the human GRCh38 (hg38) genome assembly.

About MPRA Base

MPRA Base is a curated database of experimentally tested cis-regulatory elements measured using high-throughput reporter assays. These assays quantify enhancer activity by testing thousands of candidate sequences in parallel.

The database integrates multiple assay platforms, species, and cell types, while preserving experiment-level resolution.

Track Structure

Each row represents a regulatory element tested within a specific experiment. An experiment is defined by:

  • Cell line or tissue
  • Assay type
  • Publication (PMID)

The same genomic element may appear multiple times if assayed in different experiments.

Score Normalization

  • Elements are ranked by raw activity score within each experiment.
  • A percentile rank (0–100) is computed independently per experiment.
  • A UCSC-compatible scaled score (0–1000) is derived per experiment for bed format compatibility.

The score field is included for UCSC BED compatibility. Score-based shading is not enabled in this track. Display color is determined exclusively by percentile-based RGB bins.

Color Coding

  • Blue – Percentile < 50
  • Orange – Percentile 50–74
  • Red – Percentile ≥ 75
