MPRA Base Enhancer Elements (hg38)

This track displays enhancer elements assayed using Massively Parallel Reporter Assays (MPRA), STARR-seq, and related reporter technologies across multiple published studies. All genomic coordinates are provided in the GRCh38/hg38 genome assembly.

Data Aggregation

This track aggregates elements from multiple independent MPRA experiments. Where necessary, hg19 coordinates were lifted over to hg38 using UCSC liftOver. Unmapped intervals were excluded.

Each element represents a tested regulatory sequence from a published experiment.

Definition of an "Experiment"

An experiment is defined as a unique combination of:

  • Cell line / tissue
  • Assay type (e.g., lentiMPRA, plasmidMPRA, STARR-seq)
  • PubMed ID (PMID)

Percentile ranks are computed independently within each experiment.

Percentile Rank Calculation

For each experiment:

  • Elements are sorted by raw activity score.
  • Percentile rank is computed relative to other elements within the same experiment.
  • This produces a value between 0–100.

Percentile = 0 corresponds to the lowest activity element within an experiment, and percentile = 100 corresponds to the highest activity element.

This normalization ensures comparability within experiments, while avoiding cross-experiment scale distortion (e.g., MPRA vs STARR-seq dynamic range differences).

Color Coding Scheme

Elements are colored according to percentile rank within experiment:

  • Blue: Percentile < 50 (Lower activity)
  • Orange: 50 ≤ Percentile < 75 (Intermediate activity)
  • Red: Percentile ≥ 75 (Higher activity)

Coloring reflects relative regulatory activity strength within each experiment. Visualization is controlled by RGB color values stored in the itemRgb field.

Scaled Score Field (BED Compatibility)

The UCSC BED score field (integer 0–1000) contains a linear scaling of raw activity scores within each experiment.

For each experiment:

  • Raw activity scores are linearly rescaled to a 0–1000 range.
  • The lowest raw score maps to 0.
  • The highest raw score maps to 1000.

This scaling is performed independently within each experiment and is included only for UCSC BED format compatibility.

Score-based shading (useScore) is not enabled in this track. Color visualization is determined exclusively by the percentile-based RGB bins described above.
